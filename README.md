Bookmark Manager
================

The task is to create a web app for adding and saving URL links.

Technologies used:

* Testing - Rspec, Capybara
* Back-end - Ruby
* Front-end - HTML, CSS


To download and use:
````
$ git clone https://github.com/heather-camcam/bookmark-manager-2.git
$ cd bookmark-manager-2
$ ruby app.rb
$ visit http://localhost:4567/
````
